## ğŸ® Usage

### Direct Execution
```bash

```

### Using Aliases
The script will offer to create the following aliases:
- `youdownload [URL]`
- `youtubedownload [URL]`
- `ydown [URL]`

Example:
```bash
ydown https://youtube.com/watch?v=...
```

### Command Line Arguments
- No arguments: Script will prompt for URL
- URL as argument: Starts downloading immediately
